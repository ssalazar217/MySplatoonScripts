# My Splatoon Scripts

Repositorio de scripts personalizados para el plugin Splatoon de FFXIV.

## Scripts Disponibles

### Duties / Stormblood
*   **UCOB Thunder Markers**: Marca automáticamente con Attack 1 y 2 a los jugadores afectados por Thunderstruck en el encuentro de UCOB (The Unending Coil of Bahamut).

## Instalación

1.  Copia la ruta absoluta de la carpeta `Duties` de este repositorio.
2.  En FFXIV, abre la configuración de Splatoon (`/splatoon`).
3.  Ve a la pestaña de **Settings** -> **Scripts**.
4.  Añade la ruta en la sección de **Local Scripts**.
5.  Recarga y activa los scripts deseados.
